#  SEVA - Transparent Charitable Giving on Blockchain

**SEVA** is a decentralized platform that brings **transparency and efficiency** to charitable giving using **blockchain technology**.  
Every donation is **tracked on-chain**, ensuring donors can see exactly how their contributions are used.

---

## Overview

Many charitable organizations struggle with **donor trust and transparency**. SEVA solves this by:
- Enabling **direct donations** to causes via blockchain  
- Using **smart contracts** to track every donation  
- Providing **instant, tamper-proof donation history**  
- Allowing users to **verify transactions** using Web3 wallets like MetaMask

##  Features

- **Secure donations** via Ethereum smart contracts  
- **Transparent tracking** of every contribution  
- **Cause selection** and categorization  
- **Web3 integration** with MetaMask  
- **Instant updates** on the frontend when donations occur  
- **Responsive UI** for desktop and mobile

---

## Tech Stack

- **Blockchain:** Ethereum / Solidity smart contracts  
- **Frontend:** HTML, CSS, JavaScript, React.js  
- **Web3 Integration:** MetaMask, ethers.js / web3.js  
- **Backend:** None (fully decentralized) or optional API for analytics  
- **Deployment:** IPFS / Vercel / Netlify (frontend), Ethereum testnet/mainnet (smart contracts)
